Question-1: What are the different ways to select an element in the DOM?

  1. by using the getElementById() method
  2. by using the querySelector() method
  3. by using the querySelectorAll() method
  4. by using the getElementsByClassName() method
  5. by using the getElementsByTagName() method

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: It returns the HTML content of an element including text and tags.
innerText: It returns the only text content of an element.
textContent: It returns the text content of an element ignoring all html tags inside it .

Question-3: What is event delegation in the DOM?

event delegation is a technique involving adding event listeners to a parent element instead of adding them to the invidual elements.

Question-4: What is event bubbling in the DOM?

event bubbling is a process when an event is occured on an element then it first works on that element then it moves to its parent element and so on.

Question-5: How do you create, add, and remove elements using JavaScript?

we can create elements using the createElement() method, add elements using the appendChild() method and remove elements using the removeChild() method.
